Features

Everything You Need

Built with privacy, security, and developer experience in mind.

Unified Management

Manage OpenAI, Anthropic, and custom OpenAI-compatible providers all in one place.

API Key Security

Securely store and manage multiple API keys per provider with local encryption.

Model Catalog

Detailed view of models with context windows, pricing, and capabilities.

Local Storage

All data stored locally on your machine. Your data never leaves your device.

Easy backup and migration of your configuration between machines.
